---

## 0. 方針サマリー

- 50KW を **15記事** に集約（カニバリ防止のため同義KWは1記事に統合）。
- 単体ビッグワード（「AI」「ChatGPT」など）は狙わず、**2語以上のサジェスト系**を中心に対策。
- 5つの軸（**自社専用 / オリジナル / 相談窓口 / 面倒回避 / 何から始める**）をバランス良く配置。
- 全記事の着地点はナイリーの独自価値：**好きな名前で使える・会社名/ブランド名/ロゴ・専用URL・相談窓口化・手軽さ・低コスト**。
- 過剰表現（完全自動化・全文書学習・高度なRAG・大企業向け基盤）は使わない。

### 優先順位の凡例

| 記号 | 意味 |
|---|---|
| S | 最初に作るべき（**初期5本**） |
| A | 次点で作るべき（次の10本） |
| B | 後回し・サブKW吸収でも可 |

### 制作順の考え方（READMEと連動）

初期5本は、依頼の指定（自社専用AIチャット / オリジナルAI / AI 何から始める / AI導入 めんどくさい / AI相談窓口）をそのまま採用しました。
理由：5軸を1本ずつ過不足なくカバーでき、かつナイリーのCVに近い「導入検討」と、入口になる「認知」をバランスできるためです（詳細はREADME参照）。

### A01｜自社専用AIチャット（ピラー） ★初期5本

- **対策KW**：自社専用 AIチャット
- **サブKW**：会社専用 AIチャット / 企業専用 AIチャット / 専用AIチャット / AI 自社専用 / 自社専用 AI
- **想定検索意図**：自社だけで使えるAIチャットとは何か、どう用意するか、費用や流れを知りたい。
- **読者**：AIに詳しくない中小企業の経営者・担当者
- **記事タイプ**：基礎解説（ピラー）
- **想定タイトル案**：自社専用AIチャットとは？中小企業が手軽に持つ方法と費用の考え方
- **slug案**：`/blog/company-dedicated-ai-chat`
- **CTA方針**：難しい開発なしで、会社名・ロゴ・専用URL付きの自社専用環境を小さく始められることを訴求。
- **内部リンク先候補**：A02(オリジナルAI) / A06(会社名で使う) / A03(何から始める) / A11(簡単導入) / A12(費用)
- **独自性**：「専用」を“自前サーバ構築”ではなく“会社名・ロゴ・専用URLで自社専用のように見せる”と定義し直す視点。
- **優先順位**：S（クラスタ中核ピラー）
