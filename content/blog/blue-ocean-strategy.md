血みどろの価格競争に疲弊する市場（レッドオーシャン）で戦い続けるか、それとも競争のない新しい市場（ブルーオーシャン）を切り開くか。W・チャン・キムとレネ・モボルニュが提唱したブルーオーシャン戦略は、後者を目指すための考え方です。

2005年にハーバード・ビジネス・スクール・プレスから出版された書籍『ブルー・オーシャン戦略』は、36言語に翻訳されて400万部を超えるベストセラーとなり、世界中の経営者・戦略家に影響を与えました。この記事では、<span class="highlight-yellow">ブルーオーシャン戦略の本質と実践ツール</span>を詳しく解説します。

## ブルーオーシャン戦略とは

ブルーオーシャン戦略とは、既存の競争が激しい市場（レッドオーシャン）を避け、競争自体を無意味にする新しい市場（ブルーオーシャン）を創造する戦略です。

### レッドオーシャンとブルーオーシャンの比較

| | レッドオーシャン | ブルーオーシャン |
|---|---|---|
| 市場 | 既存市場・競争激化 | 新規市場・競争なし |
| 戦い方 | 競合を打ち負かす | 競争を無意味にする |
| 価値とコスト | トレードオフ（二律背反） | 両立を目指す |
| 顧客 | 既存顧客を奪い合う | 非顧客を顧客に変える |
| 目的 | 市場シェアの拡大 | 新しい需要の創造 |

「レッドオーシャン」という名前は、競合同士が血みどろの戦いを繰り広げ、海が血で赤く染まるイメージから来ています。「ブルーオーシャン」は、広く澄んだ未開の海を表します。

<div class="callout-info"><p>💡 ポイント: ブルーオーシャン戦略は「競合を無視する」ことではありません。「競合と同じ軸で戦うことをやめる」ことです。業界の常識を疑い、顧客が本当に求めている価値を再定義することがスタートポイントです。</p></div>

## バリューイノベーション

ブルーオーシャン戦略の核心が「バリューイノベーション（価値革新）」です。

### バリューイノベーションとは

通常の経営では「価値を上げればコストも上がる（差別化コスト）」「コストを下げれば価値も下がる（コストリーダーシップ）」というトレードオフが前提です。

バリューイノベーションでは、**価値を上げつつコストも下げる**ことを同時に目指します。これを可能にするのが、「業界の常識として当たり前にやっていること」を見直すことです。

### ポーターとの違い

ポーターの競争戦略（→ [ポーターの競争戦略](/blog/porter-competitive-strategy)）では「コストか差別化か」という選択を迫りますが、ブルーオーシャン戦略では「どちらも同時に追求できる」と考えます。

| | ポーター競争戦略 | ブルーオーシャン |
|---|---|---|
| 基本姿勢 | 競合との競争で勝つ | 競争を無意味にする |
| 価値とコスト | トレードオフ | 同時追求 |
| 市場 | 既存市場 | 新市場の創造 |

## 戦略キャンバスとERRCグリッド

ブルーオーシャンを描く2つの主要ツールがあります。

### 戦略キャンバス（Strategy Canvas）

業界の競争要因を横軸に、各社の提供レベルを折れ線グラフで描き、自社が競合と同じ形になっていないかを可視化します。

**作成手順**

1. 業界の主要な競争要因を横軸に並べる（価格・品質・スピード・デザイン・サポートなど）
2. 競合他社の各要因のレベルを折れ線で描く
3. 自社の折れ線を描く
4. 競合と自社の折れ線が似ていないか確認する

**読み方**
- 競合と自社が同じような曲線を描いている→レッドオーシャンにいる可能性が高い
- 独自の「突出」と「低下」がある→差別化できている可能性がある

**例：外食業界の戦略キャンバス**

横軸：価格・立地・品質・メニューの多様性・スピード・雰囲気・サービス

ファストフードチェーンは「価格・スピード」が高く「雰囲気・サービス」が低い。高級レストランはその逆。もし「スピードと雰囲気を両立」した価格帯を作ると、ブルーオーシャンになりえます。

### ERRCグリッド（アクションマトリクス）

<span class="highlight-yellow">ERRCグリッドは、バリューイノベーションを実現するための4つの行動指針</span>です。

| アクション | 問い | 目的 |
|---|---|---|
| **Eliminate（取り除く）** | 業界で当たり前だが、本当に必要かわからないものは？ | コストを削減し、過剰な要素を除く |
| **Reduce（減らす）** | 業界標準より削れる要素は？ | コストを下げ、価値低下を最小化 |
| **Raise（増やす）** | 業界標準より大幅に高めるべき要素は？ | 顧客にとっての価値を高める |
| **Create（創造する）** | 業界にまだない、新しく加える要素は？ | 競合にない新たな価値を生む |

E・Rでコストを下げ、R・Cで価値を高める。これでバリューイノベーションを実現します。

**ERRCグリッドの活用例：ネットフリックスの初期戦略（仮説）**

| アクション | 内容 |
|---|---|
| Eliminate | 物理的な店舗・延滞料 |
| Reduce | 在庫管理コスト（デジタル配信で不要化） |
| Raise | コンテンツのアクセシビリティ（いつでもどこでも） |
| Create | 月定額サブスクリプションモデル |

この分析により、レンタルビデオ業界の常識を覆し、ブルーオーシャンを創造しました。

## ブルーオーシャンの6つのパス

ブルーオーシャン戦略では、新市場を発見するための「6つのパス（視点の変換）」を提示しています。

**パス1：代替産業を見る**
異なる産業の製品が同じニーズを満たしている場合、その間に新市場がある可能性があります。
例：タクシーとバスの間の「ライドシェア」

**パス2：業界内の戦略グループを見る**
同業内の高価格帯・低価格帯の間に、ニーズが満たされていない層がいないか。
例：高級ホテルとビジネスホテルの間の「ブティックホテル」

**パス3：バイヤーチェーンを見る**
購買者（Buyer）・使用者（User）・影響者（Influencer）を分けて考える。通常と異なる層にフォーカスすると新市場が見える。
例：従来は医師向けに売っていた製品を患者（使用者）向けに設計し直す

**パス4：補完製品・サービスを見る**
自社製品と合わせて使われるものが提供できていない価値を補完する。
例：IKEAが家具だけでなく「組み立て体験」「インテリアコーディネート」も提供

**パス5：機能・感情の軸を見る**
機能的価値で戦っている業界に感情的価値を追加、またはその逆。
例：機能重視の業務ソフトに「使っていて楽しい」体験を追加

**パス6：時間の軸を見る**
社会・技術・生活様式の変化の中に、将来の新需要を見る。
例：高齢化社会に向けた新サービス・5G普及後の新サービス

<div class="callout-tip"><p>✅ 実践ポイント: ERRCグリッドを使う際は、最初に「業界の常識リスト」を作ることをお勧めします。「業界ではこれが当たり前」「皆がやっていること」を10〜20個書き出し、「本当に必要か？」と問い直すところから始めましょう。</p></div>

## 非顧客に目を向ける

ブルーオーシャン戦略で重要な視点が「非顧客（Non-Customer）」に注目することです。

**非顧客の3層**

| 層 | 内容 | アプローチ |
|---|---|---|
| 第1層：将来の顧客 | 最低限使っているが、機会があれば離れたい | 満足度の向上・価値改善 |
| 第2層：拒絶した人 | 意識的に使わないと選択した人 | 「なぜ使わないか」の理由に新市場がある |
| 第3層：未探索の人 | 存在を知らない・全く縁のない人 | 全く新しいアプローチが必要 |

**活用方法**

「今使っていない人が、なぜ使っていないか」「使うとしたら何が変われば使うか」を深く理解することで、既存のサービスに不満がある「潜在市場」を発見できます。

## 既存のフレームワークとの関係

ブルーオーシャン戦略は、単独で使うより他のフレームワークと組み合わせることで力を発揮します。

- **ファイブフォース分析との関係**：5つの力が強い業界（レッドオーシャン）を分析した後、競争を弱めるためにブルーオーシャン戦略を検討（→ [ファイブフォース分析とは](/blog/porter-five-forces)）
- **ポジショニングとの関係**：新しい価値軸で独自の立ち位置を作る点で共通（→ [カテゴリーデザインとは](/blog/category-design)）
- **カスタマージャーニーとの関係**：顧客が「なぜその体験が不満か」を理解することがERRCの出発点

## 実践のポイントと注意点

### 競合をベンチマークしすぎない

競合を詳しく調べることは重要ですが、競合に「合わせていく」方向になると、結局レッドオーシャンに戻ります。競合分析は「どこが戦場になっているか」を知るためで、「どう同じ土俵で戦うか」を決めるためではありません。

競合の動向をモニタリングしながらも、「同じ軸で戦わない」という姿勢を保つことが重要です。競合の発表・プレスリリース・SNSを継続的に追う [ReAnker](/) のようなツールと組み合わせると、「業界が向かっている方向」を把握しながら「違う方向を探す」戦略思考に活かせます（月額300円、無料プランあり）。

### ブルーオーシャンは永遠でない

<span class="highlight-yellow">成功したブルーオーシャンは、競合の模倣によっていずれレッドオーシャン化します</span>。UberやAirbnbも、登場当初はブルーオーシャンでしたが、現在は多数の競合が参入し競争が激化しています。

- ブルーオーシャンを作ったら、次のブルーオーシャンを探し続ける
- 競合の追随を定期的にモニタリングする
- 「競争優位の持続期間」を意識した戦略を持つ

### 顧客のニーズを出発点に

ERRCグリッドは「業界の論理」から発想しますが、最終的には「顧客にとって何が価値か」に立ち戻ることが重要です。業界の非常識が「顧客にとって価値がない」なら、それはブルーオーシャンではなく単なる品質低下です。

<div class="callout-warn"><p>⚠️ 注意: 「ブルーオーシャンを作る」という目的で動くより、「顧客が今まだ解決されていない不満・問題はないか」という問いから始めることをお勧めします。市場の論理からではなく、顧客の体験から始めることが、本物のブルーオーシャンへの近道です。</p></div>

## ブルーオーシャン戦略の実例（概要）

### Cirque du Soleil（シルク・ドゥ・ソレイユ）

書籍で最も有名な事例です。

**従来のサーカス業界の常識**
- 動物の芸・スター演者・複数リング・大衆向け

**シルク・ドゥ・ソレイユのERRC**
- Eliminate：動物・スター演者・複数リングショー
- Reduce：面白おかしさ・スリル・危険
- Raise：会場の快適さ
- Create：テーマ・芸術的音楽・ダンス・演劇的ストーリー

**結果**：競合のいない「大人向けのアート型サーカス」という新ジャンルを創造。高価格帯でも客が集まるブルーオーシャンを実現。

### Nintendo Wii（任天堂）

ゲーム業界でのブルーオーシャン創造の事例です。

**従来のゲーム機競争**
- 処理能力・グラフィック・ハードコアゲーマー向け

**WiiのERRC**
- Eliminate：高性能グラフィック処理
- Reduce：ゲームの複雑さ
- Raise：体を使う直感的操作
- Create：家族・高齢者が一緒に遊べる体験

**結果**：「ゲームをしない人（非顧客）」を顧客に変え、ゲーム市場を大幅に拡大しました。

## まとめ

ブルーオーシャン戦略は、競争の激しい市場を避け、価値とコストを両立させるバリューイノベーションで新市場を創造する戦略です。ERRCグリッドと戦略キャンバスを使って業界の常識を問い直し、競争を無意味にする独自の価値を設計しましょう。

ただし、切り開いた市場もいずれ模倣される点には注意が必要です。ブルーオーシャンは「一度作れば終わり」ではなく、継続的なイノベーションの姿勢が求められます。
